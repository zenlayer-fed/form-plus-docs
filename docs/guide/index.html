# 介绍

form-plus 是一个表单生成器，主要通过一个 js-json 来快速生成表单，在重复表单创建上，有具体的应用场景，使使用者只需关心配置即可。

# 安装

$ npm i @zenlayer/form-plus -D

# 配置

支持 cjs/esm 形式，生成 ES module 能够更好地进行 tree shaking。example

form-plus 依赖 zen-ui 基础组件库，请保证在项目中正确使用zen-ui,如未使用 zen-ui，可直接引入/dist/lib.js,但不建议这么操作，因为你会引入近 2M 的代码。

# V1 至 V2

v2 中重构了整体架构，支持 es module，优化了整体大小 💯，这是在 v1 中不支持的。
