    # Api

    key value You need to know
    [Item.id].data Obj select 等组件的数据源，其中 disabled=boolean | (form) => boolean
    [Item.id].defaultValue any 默认值
    [Item.id].external string[] 同期在 form 中存储的 key
    [Item.id].remote.get "get" 请求方式
    [Item.id].remote.api string 支持${[item.id]}模版，会在对应 id 的 value 变化时发送请求
    [Item.id].remote.path string res 数据管道
    [Item.id].remote.format Obj[] 数据格式化，对应字断的取值
    [Item.id].remote.dataFormatFn (data,form)=>Obj[] 数据格式化
    [Item.id].remote.requestCustom (...other)=>Ajax(...other) 自定义请求器
